# Docker Practice: Day 02

This folder is a practice implementation of the official Docker sample application. 

## Project Overview
This repository is a sample application for users following the getting started guide at [https://docs.docker.com/get-started/](https://docs.docker.com/get-started/).

The application is based on the application from the getting started tutorial at [https://github.com/docker/getting-started](https://github.com/docker/getting-started).

---

## Learning Objectives
* Understanding **Dockerfiles** and image layers.
* Practicing container lifecycle management (`build`, `run`, `stop`).
* Exploring multi-container setups and networking.

> **Note:** This is a local practice environment and is not intended for production use.
